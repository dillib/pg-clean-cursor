Why this is the recommended pick
  • Every text pairing has a documented contrast ratio — no guessing, no A11y surprises in audit.
  • Tokens named by role (fg-muted, ui-border), not by value — makes it obvious which to reach for.
  • Solid values render identically on white, cream, yellow, and in exported PDFs.
  • Warm cast preserved — still feels like PhotonicTag, not Stripe or Linear.
  • Scale is wide enough (11 steps) to compose a modal-on-panel-on-page without two surfaces colliding.
Migration from the current alpha system
  • ink → n-11
  • ink-72 / fg-muted → n-9
  • ink-56 → n-8
  • ink-40 / fg-subtle → n-7 (and only on large UI)
  • ink-12 / hairline → n-4
  • paper stays paper; page becomes n-1
